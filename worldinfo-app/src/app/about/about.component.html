O projekcie WorldInfo

WorldInfo to jednostronicowa aplikacja (SPA) napisana w Angularze, która umożliwia szybkie przeglądanie informacji o krajach świata.

TECHNOLOGIE

  • Angular 15+ (standalone components)
  • REST Countries API v3.1
  • Bootstrap 5 + Font Awesome
  • SCSS + Responsive Design
  • Leaflet (mapa interaktywna)

FUNKCJONALNOŚCI

  • Przeglądanie listy krajów w formie kart
  • Wyświetlanie szczegółów kraju po kliknięciu kraju na mapie
  • Interaktywna mapa — kliknięcie w kraj przenosi do jego szczegółów
  • Responsywność na różnych urządzeniach
  • Routing bez przeładowań strony

ŹRÓDŁA DANYCH

Dane pochodzą z otwartego API: https://restcountries.com
